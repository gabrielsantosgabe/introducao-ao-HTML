Estrutura Básica do HTML

  1. < html> < /html> - Define a raiz de um documento HTML

  2. < head> < /head> - Contém informações para o documento

  3. < body> < /body> - Define o corpo do documento

  4. < title> < /tile> - Define o titulo do documento

Falando sobre Tags

  1. < i> < /i> - É usada para deixar o texto em itálico

  2. < strong> < /strong> - Deixa o texto em negrito

  3. < input type=type"text"> - Te da um campo para digitar letras

  4. < input type=type"number"> - Te da um campo para digitar numeros

  5. < input type=type"color"> - Te da um campo para escolher cores
